
# 📊 Superstore Sales Analysis Dashboard (Microsoft Excel)

## Project Overview

This project analyzes Superstore sales data using Microsoft Excel to uncover business insights and build an interactive dashboard. The dashboard enables users to explore sales performance across regions, product categories, customer segments, and shipping modes using interactive slicers.

The objective of this project is to demonstrate practical Excel skills including data cleaning, data validation, Pivot Tables, Pivot Charts, dashboard design, and business analysis.

## Key Insights

- West region generated the highest sales.
- Technology contributed the highest share of profit around 51%.
- Standard Class was the most frequently used shipping mode.
- Chairs and Storage were among the highest-selling sub-categories.
- Higher discounts were associated with reduced profitability in several cases.

---

## Business Recommendations

- Increase marketing investment in high-performing regions.
- Review discount strategies to improve profit margins.
- Promote high-margin Technology products.
- Optimize inventory for best-selling sub-categories.
- Investigate underperforming regions and categories for growth opportunities.
